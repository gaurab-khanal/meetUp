# meetUp -> A peer to peer video calling app.

## Tech stack
- NextJS
- NodeJS
- Express
- WebRTC
- SocketIo# meetUp


## Setup

 ### Client Setup
    
    ```
        cd client
        npm i
        npm i -g peer
        npm run dev
        peerjs --port 3001 // in another terminal
    ```



### Server setup

    ```
    cd server
    npm i 
    npm run dev
    ```
